# 🍕 FreshBite - Online Food Ordering Website

## 📌 Project Description
FreshBite is a responsive online food ordering website developed using HTML, CSS and JavaScript.

## 🚀 Features
- Responsive Home Page
- Menu Page
- Restaurant Listing
- Shopping Cart
- Checkout Page
- Login Page
- About Us
- Contact Us
- Add to Cart
- Cart Counter
- Total Price Calculation

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
